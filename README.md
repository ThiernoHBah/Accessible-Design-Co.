# Accessible Design Co. Webpage

This project is a simple homepage for **Accessible Design Co.**,
showcasing the brand mission and basic layout features. The page
includes:

-   A navigation bar with links (Home, About Us, Services, Contact)
-   Login and password buttons
-   A welcome section explaining the company purpose
-   A four box grid layout to organize information visually

The site is styled using CSS to create a modern and clean layout.

## How to View or Run This Project

1.  Download the project files (`index.html` and `styles.css`).
2.  Make sure both files are in the same folder.
3.  Double‑click `index.html` to open it in your web browser.
4.  The page will load and display the design and text automatically ---
    no extra setup needed.
